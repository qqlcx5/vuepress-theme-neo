## 详情

V2 基于 _VuePress2_， 带有 _Vite2_ / _Webpack5_ 和 _Vue3_ 的强大功能。

V2 完全重写，包括:

- 所有组件都用 _Vue3 中的 Composition API_ 重写
- 所有样式都迁移到 _Sass_

V2 也有更好的性能 🚀:

- DevServer 冷启动时间减少 **50%**
- 构建内存减小 **65%**
- 构建时间减少 **30%**
- 输出大小降低 **25%**
- 网页性能提高 **70%**
