# shuffle 打乱数组

创建一个打乱的数组，使用Fisher-Yates算法打乱数组的元素

shuffle(array)

参数

  • array 要打乱的数组

例子

shuffle([2, 3, 1])
// => [3, 1, 2]

源码

const shuffle = ([...arr]) => {
    let m = arr.length
    while (m) {
        const i = Math.floor(Math.random() * m--);
        [arr[m], arr[i]] = [arr[i], arr[m]]
    }
    return arr
}
